# WEB103 Unit 4 Project - Bolt Bucket

Bolt Bucket is a custom car builder made with React, Express, and PostgreSQL. Users can choose a body style, paint color, wheels, interior, and package, then save their build to the database.

## Setup

1. Run `npm install`
2. Add your Render PostgreSQL values to `server/.env`
3. Run `npm run reset` to create and seed the `custom_cars` table
4. Run `npm run dev`
